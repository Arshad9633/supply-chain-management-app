# Supply Chain Management System

A full-stack application to manage suppliers, products, inventory, and orders.

## Tech Stack

### Backend
- Java
- Spring Boot
- MongoDB
- REST APIs
- Swagger (OpenAPI)

### Frontend
- React (Vite)
- Axios
- React Router

## Features

### Supplier Management
- Create supplier
- View supplier list
- Update supplier
- Delete supplier

### Product Management
- Create product linked to supplier
- View products
- Filter by supplier

### Inventory Management
- Track stock per warehouse
- Prevent duplicate inventory entries
- Low stock detection

### Order Management
- Create orders
- Reduce stock automatically
- Prevent over-ordering
- Track order status
